Die Anwendung

Die App dient der Übersetzung eines Bildes. Ein Foto wird per Handykamera aufgenommen und für den Benutzer übersetzt. Den Text kann der Nutzer der App danach ganz einfach ablesen.

Die App wird nicht im Altag verwendet, sie dient der Erleichterung von Reisen. Da die Fremdsprachen welche im Zielland gesprochen werden, oft nicht von den Nutzern verstanden werden.


Tiefer in die Materie

Wenn die App startet, hat der Nutzer die Möglichkeit ein Bild auszuwählen, der ein neues Foto aufzunehmen. Danach wird das ausgewählte/aufgenommene Foto an einen von uns erstellten Webserver gesendet. Auf diesem Webserver läuft dann die Zeichenerkennungssoftware (und die Übersetzung wird durchgeführt). Diese Software filtert sich die Buchstaben aus dem Bild heraus. Diese werden dann (wieder an die App gesendet und dann per Microsoft Cognitive Services übersetzt.) (direkt am Webserver per Microsoft Cognitive Services übersetzt, der übersetzte Text wird dann wieder auf das Handy gesendet und der Nutzer hat einen Text vor sich.)

How it works

Picturelate einfach erklärt

Take a Picture

Wie in der jeden bekannten Kamerafunktion des Smartphones können sie ein Bild aufnehmen welches sie übersetzen wollen.

Oder sie wählen ganz einfach eines aus Ihrer Galerie aus.

Read and Tranlate the Picture

Das Bild wird an einen Webserver gesendet, durch eine Software wird der Text aus dem Bild heraus analysiert. Danach wird (dieser wieder an die App gesendet und per Microsoft Cognitive Services übersetzt.) (dieser direkt am Webserver per Microsoft Cognitive Services übersetzt, der übersetzte Text wird dann wieder auf das Handy gesendet)

Show the Text

Nachdem das Foto aufgenommen wurde, muss der Nutzer nur noch kurz warten. Nach einigen wenigen Sekunden wird ihm der fertige übersetzte Text angezeigt
